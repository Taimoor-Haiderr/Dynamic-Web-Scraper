#  Dynamic Web Scraper (Tkinter GUI)

A powerful and user-friendly desktop application for scraping **Jobs, Products, and News** data from websites in real time.

---

##  Features

*  Scrape **Jobs, Products, and News**
*  Works with:

  * HTML pages
  * JSON APIs
  * RSS/XML feeds
*  Multi-page scraping support
*  Fast & responsive (threaded scraping)
*  Live search and filtering
* Sortable data table
* Export data to:

  * JSON
  * CSV
  * Excel (.xlsx)
*  Activity log with real-time updates
* Modern dark-themed UI


## Tech Stack

* Python
* Tkinter (GUI)
* Requests (HTTP requests)
* BeautifulSoup (HTML/XML parsing)
* Pandas (Excel export)


##  Usage

1. Select a category:

   * Jobs
   * Products
   * News

2. Paste a target URL

3. Choose number of pages to scrape

4. Click **"Scrape Data"**

5. View results in the table

6. Use:

   * Search bar to filter data
   * Sort buttons for ordering

7. Export results using:

   * Save JSON
   * Save CSV
   * Export Excel


##  Supported Data Extraction

### Jobs

* Title
* Company
* Location
* Salary
* Date
* Apply Link

## Products

* Name
* Price
* Rating
* Category
* Description
* Product Link

###  News

* Title
* Source
* Published Date
* Description
* Article Link
